Инициализация репозитория
	git init
	git add . (добавляем все файлы)
	git commit -m "initial" (базовый комит, который инициализирует репозиторий)
	git remote add origin (пишем ссылку на репозиторий)
	git push -u origin master

	git checkout -b "webpack" (создает и реализовывает проект в отдельной ветке)
	git add .
	git commit -m 'webpack config'
	git push -u origin webpack

Настройки webpack
	npm install webpack webpack-cli --save-dev (savedev все для разработки, в прод не идут)


Перед тем как вводить команду "npm i", обновите версии пакетов в файле package.json. Делается это так:
	1.npm i -g npm-check-updates
	2.ncu 
	3. ncu -u (или другой ключ. после "ncu" будет написано, какой ключ вводить)
	4. npm i 

очистить кеш нпм модулей
npm cache clean --force
устанавливаем заного нпм
npm i npm -g
